# 2.Главная страница: Проверить сортировку «По цене»
### TC-201: Сортировка по возрастанию цены
**Предусловие:**
Открыта главная страница  `(/, /list)`. На странице есть минимум 3 объявления с ценами.

**Шаги:**
1. Найти выпадающий список сортировки
2. Выбрать «Сортировать по: цене», «Порядок: по возрастанию»

**Ожидаемый результат:**

Цена каждого следующего объявления ≥ предыдущего.
Все объявления отображаются (количество не изменилось)

### TC-202: Сортировка по убыванию цены
**Предусловие:**
Открыта главная страница  `(/, /list)`. На странице есть минимум 3 объявления с ценами.

**Шаги:**
1. Найти выпадающий список сортировки
2. Выбрать «Сортировать по: цене», «Порядок: по убыванию»

**Ожидаемый результат:**

Цена каждого следующего объявления ≤ предыдущего.
Все объявления отображаются (количество не изменилось)

### TC-203: Сортировка с примененным фильтром
**Предусловие:**
Открыта главная страница  `(/, /list)`. На странице есть объявления с разной ценой и категорией.

**Шаги:**
1. Найти выпадающий список «Категория»
2. Применить фильтр категории (например, «Электроника»)
3. Найти выпадающий список сортировки
4. Выбрать «Сортировать по: цене», «Порядок: по убыванию»

**Ожидаемый результат:**

1. Сортировка применяется только к отфильтрованным объявлениям
2. Порядок цен корректный в рамках фильтра
