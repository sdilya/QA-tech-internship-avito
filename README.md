# Инструкция по запуску автотестов
Автотесты написаны на Python и используют библиотеку pytest. 

### Установка зависимостей

Выполните в корне проекта:
```bash
pip install -r requirements.txt
```
# API-tests(2.1)
### Запуск тестов

Выполните команду:
```bash
pytest avito-api-tests/ -v
```
### Дополнительно

**Только E2E-сценарии**

Выполните команду:
```bash
pytest avito-api-tests/ -v -m e2e
```
**Проверка и форматирование кода**

Проверить код на ошибки стиля:
```bash
flake8 avito-api-tests/
```
Автоматически отформатировать код:
```bash
black avito-api-tests/
```
Проверить, что код отформатирован:
```bash
black --check avito-api-tests/
```
# UI-tests(2.2)
### После установки зависимостей нужно скачать браузеры для UI-тестов:
```bash
playwright install chromium
```
## Примечание по запуску UI-тестов

Тестовый стенд `https://cerulean-praline-8e5aa6.netlify.app` может быть доступен только из определённых сетей или требовать VPN.

### Если тесты не запускаются:
1. Убедитесь, что сайт открывается в браузере
2. При необходимости включите VPN
3. Или оцените решение по коду и тест-кейсам
   
### Запуск тестов

Выполните команду:
```bash
pytest avito-ui-tests/ -v
```

В видимом режиме (показать браузер)
```bash
pytest avito-ui-tests/ -v --headed
```
